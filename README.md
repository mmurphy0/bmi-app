# BMI calculator

## Description
An app made using Python tkinter which allows users to see if their bmi is healthy or not. Also, users can save their results to records.txt so they can track their results over time

## Usage
-> User is welcomed with the startup window which has options to measure BMI or Exit

-> The Measure BMI Button Leads onto the Personal Information where the user must enter their height in metres and weight in kg. There is a submit button to let the program calculate you BMI and return the results to you. Also , there is a back button to let users go back to the startup window if they wish to.

-> The next window is the results window, it shows you BMI and tells you if you BMI is in the underweight, healthy weight, overweight or obese categories.

-> Then the user must click the save button to save their results to records.txt
